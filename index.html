Game Guide: How to Play

In Desktop

  1. Drag & Drop: Click and drag a colored box into an empty slot.
  2. Submit Your Guess: Click the "Check Guess" button to check if your combination is correct.
  3. Hint System:
    • Black ✔ = Correct number in the correct position.
    • White ⚪ = Correct number in the wrong position.
    • Red ❌ = Incorrect number.
  4. Continue: Continue guessing until you crack the code!
  5. Winning Condition: If all numbers are in the correct position, you win.
  6. Restarting: Click the "Restart" button to start a new game with a new secret code.

In Mobiles

  1. Tap & Select: Tap a colored box to select it. It will be highlighted.
  2. Tap to Place: Tap an empty slot to place the selected box there.
  3. Submit Your Guess: Click the "Check Guess" button to check if your combination is correct.
  4. Hint System:
    • Black ✔ = Correct number in the correct position.
    • White ⚪ = Correct number in the wrong position.
    • Red ❌ = Incorrect number.
  5. Continue: Continue guessing until you crack the code!
  6. Winning Condition: If all numbers are in the correct position, you win.
  7. Restarting: Click the "Restart" button to start a new game with a new secret code.
